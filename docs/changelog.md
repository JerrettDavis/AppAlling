## [0.1.0] - 2025-09-10

### Added

- Initial public release of **AppAlling**
- Command Bus for dispatching application actions
- Reactive State Store with `IStore<T>`
- MenuComposer with data-driven menu generation
- Plugin loader with discovery and DI registration
- Sample HelloWorld plugin with command, menu, and tool window contributions
- End-to-end tests and coverage reporting

# Contributing

We ❤️ contributions! Whether you are fixing a bug, improving documentation, or adding new features, your help makes AppAlling better for everyone.

## Ways to Contribute

- **Report issues** – Found a bug or have a feature request? [Open an issue](https://github.com/JerrettDavis/AppAlling/issues/new).
- **Improve docs** – Fix typos, add examples, clarify confusing sections.
- **Submit PRs** – Tackle open issues or propose enhancements with pull requests.
